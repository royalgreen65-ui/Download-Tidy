# File-Zen Setup Instructions

## Prerequisites
- Node.js installed

## Setup
1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Variables**
   - Create a `.env` file in the root directory.
   - Add your Gemini API key:
     ```
     GEMINI_API_KEY=your_api_key_here
     ```

3. **Run Application**
   ```bash
   npm run dev
   ```
   or open `run.bat` on Windows.

## Notes
- The application uses the Gemini API for file categorization.
- Ensure you have a valid API key.
